Класс MessageController является контроллером для обработки HTTP-запросов.

sendMessage(String) - Этот метод обрабатывает POST-запросы
     по адресу /messages, содержащие в себе текст. Выводит полученный текст в консоль и также
     возвращает клиенту, записывая все полученные текстовые сообщения в лист messages.
getMessages() - Этот метод обрабатывает GET-запросы по адресу /messages.
     Пытается создать файл с сообщениями .csv на сервере, и затем отправляет его клиенту. Если до этого
     ни одного сообщения не было, возвращает клиенту статус INTERNAL_SERVER_ERROR.
List<String> messages - Хранит в себе все полученные сообщения

Проверялось с помощью Postman, где отправлялись POST и GET запросы по адресу: http://localhost:8080/messages
